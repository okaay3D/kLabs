* **Core Stack:** TypeScript/JavaScript (Angular, NestJS).
* **Purpose-made Utilities:** I develop custom tools and automation solutions, primarily in Python, to address specific workflow and productivity challenges.

### Research & Current Focus
I am currently working on my Master's Thesis, titled **"Conceptualization and Evaluation of an Event-Driven Architecture for Semantic Conflict Resolution in Offline-First Mobile Applications."** 
This research explores:
* **Event Sourcing & Vector Clocks** for tracking and resolving concurrent data modifications.
* Replicating state across decoupled systems using an **Outbox pattern**.
* Implementing reactive, responsive frontends using **Angular** for robust offline capability.

## 🛠️ The Homelab

Outside of pure software development, I am interested in self-hosting. My homelab is my sandbox for testing enterprise-grade infrastructure configurations on a consumer scale.

* **Hypervisors & Virtualization:** Running a cluster of Proxmox VE nodes utilizing lightweight LXC containers and VMs for isolated services.
* **Storage:** A custom-built, low-power NAS system powered by an Intel i3-14100 running OpenMediaVault.
* **Automation & Orchestration:** Services deployed through highly modular Docker-Compose stacks.
* **Power Redundancy:** Infrastructure protected and gracefully managed using Network UPS Tools (NUT).
* **Media Pipelines:** Automated workflows for digitizing, OCR-renaming, subtitle-based renaming, and organizing physical media collections, all served via Jellyfin.
